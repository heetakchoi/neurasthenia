Neurasthenia 가 동작하고 있습니다.

Neurasthenia 는 Java NIO 기반의 네트워크 서버로서

  • Comet 지원
  • Servlet container (Servlet, JSP 지원)
  • HTTP, STOMP 프로토콜 지원
  • Connection handling
등의 기능을 가지고 있습니다.

예제)

  • Servlet 예제
  • JSP 예제
  • Comet (Streaming) 채팅 예제

Neursathenia 는 LGPL v3. 기반의 오픈소스 프로젝트의 결과물이며 좀 더 자세한 기능 리스트는 매뉴얼 을 참조하십시오.
